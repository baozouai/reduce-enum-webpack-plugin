## About

A Webpack Plugin to reduce ts enum to js artifacts

eg:

before add pluginï¼š
```ts
enum Status {
  PAID,
  UN_PAID
}

// =>

var Status;
(function (Status) {
  Status[Status.PAID = 0] = 'PAID'
  Status[Status.UN_PAID = 1] = 'UN_PAID'
})(Status || (Status = {}))
```

after add pluginï¼š
```ts
enum Status {
  PAID,
  UN_PAID
}

// =>

var Status = {
  PAID: 0,
  UN_PAID: 1
}
```
